# touch

- `touch` create new file in current directory
  - `<path>`
  - `<fileName.extension>`

```bash
touch text.txt
```

```bash
touch text.txt blue.pdf orange.py
```

- `touch + <path> / <fileName.extension>` create a file in a specific location

```bash
touch Desktop/TerminalTutorial/text2.txt
```
