# Deploy

## IFramePlayer & AdminPortal

- deploy sul branch di test, merge di development or another branch
- `powershell` ↷

```powershell
git checkout test; git pull; git merge development; git push; git checkout development
```

- `git bash` ↷

```bash
git checkout test && git pull && git merge development && git push && git checkout development
```
